# 2pdf

Command line tool for creating PDFs from HTML files.

Based on [puppeteer.](https://github.com/puppeteer/puppeteer)

### Install

```sh
npm i -g 2pdf
```

### Usage

```sh
2pdf input.html output.pdf
```

### Troubleshooting

On Debian systems you need `libnss3` installed:

```
apt install libnss3
```
